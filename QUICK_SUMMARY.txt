📞 TROUBLESHOOTING
================================================================================

"I don't see install icon"
  → Make sure URL is HTTPS (not HTTP)
  → Wait 2-3 seconds for page to load
  → Try Chrome or Edge (Safari different)

"Service Worker not active"
  → Normal on localhost! Works on production.
  → Check DevTools → Application → Service Workers

"App won't install"
  → Must be HTTPS (localhost is OK)
  → Try a different browser
  → Clear cache and reload

"Prices not updating"
  → Provider website might have changed
  → Check console (F12) for errors
  → Manually update price in edit form
